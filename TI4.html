    Section 7.4 TI4: Integration by Partial Fractions

    Objectives

    TI4
    I can use partial fraction decomposition as a means to compute integrals of rational or rational-like functions.

    Pre-Class Preparation.

    Read the following sections of the text:
    • Section 3.4: Partial Fractions
      Read all sections

    Subsection 7.4.1 In-Class Activities

    Activity 7.4.1.

    Consider \(\displaystyle \int \frac{x^2+x+1}{x^3+x} \,dx\text{.}\) Which substitution would you choose to evaluate this integral?
    1. \(\displaystyle u=x^3\)
    2. \(\displaystyle u=x^3+x\)
    3. \(\displaystyle u=x^2+x+1\)
    4. Substitution is not effective

    Activity 7.4.2.

    Using the method of substitution, which of these is equal to \(\displaystyle\int \frac{5}{x+7} dx\text{?}\)
    1. \(\displaystyle 5\ln|x+7| +C\)
    2. \(\displaystyle \dfrac{5}{7}\ln|x+7| +C\)
    3. \(\displaystyle 5\ln|x|+5\ln|7|+C\)
    4. \(\displaystyle \dfrac{5}{7}\ln|x|+C\)

    Observation 7.4.1.

    To avoid repetitive substitution, the following integral formulas will be useful.
    \begin{equation*} \int\dfrac{1}{x+b}\, dx=\ln|x+b|+C \end{equation*}
    \begin{equation*} \int\dfrac{1}{(x+b)^2}\, dx=-\dfrac{1}{x+b}+C \end{equation*}
    \begin{equation*} \int\dfrac{1}{x^2+b^2}\, dx=\dfrac{1}{b}\arctan\lrpar{\dfrac{x}{b}}+C \end{equation*}

    Activity 7.4.3.

    Which of the following is equal to \(\displaystyle\frac{1}{x}+\frac{1}{x^2+1}\text{?}\)
    1. \(\displaystyle \frac{2x}{x^2+x+1}\)
    2. \(\displaystyle \frac{x^3+x}{x^2+x+1}\)
    3. \(\displaystyle \frac{2x}{x^3+x}\)
    4. \(\displaystyle \frac{x^2+x+1}{x^3+x}\)

    Activity 7.4.4.

    Based on the previous activities, which of these is equal to \(\ds \int \dfrac{x^2+x+1}{x^3+x} dx\text{?}\)
    1. \(\displaystyle \ln|x|+\arctan(x)+C\)
    2. \(\displaystyle \ln|x^2+x+1|+C\)
    3. \(\displaystyle \ln|x^3+x|+C\)
    4. \(\displaystyle \arctan(x^3+x)+C\)

    Activity 7.4.5.

    Suppose we know
    \begin{equation*} \dfrac{10x-11}{x^2+x-2}=\frac{7}{x-1}+\frac{3}{x+2}\text{.} \end{equation*}
    Which of these is equal to \(\ds \int\dfrac{10x-11}{x^2+x-2}\, dx\text{?}\)
    1. \(\displaystyle 7\ln|x-1|+3\arctan(x+2)+C\)
    2. \(\displaystyle 7\ln|x-1|+3\ln|x+2|+C\)
    3. \(\displaystyle 7\arctan(x-1)+3\arctan(x+2)+C\)
    4. \(\displaystyle 7\arctan(x-1)+3\ln|x+2|+C\)

    Observation 7.4.2.

    To find integrals like \(\ds \int \dfrac{x^2+x+1}{x^3+x} dx\) and \(\ds \int \dfrac{10x-11}{x^2+x-2} dx\text{,}\) we’d like to decompose the fractions into simpler partial fractions that may be integrated with these formulas
    \begin{equation*} \int\dfrac{1}{x+b}\, dx=\ln|x+b|+C \end{equation*}
    \begin{equation*} \int\dfrac{1}{(x+b)^2}\, dx=-\frac{1}{x+b}+C \end{equation*}
    \begin{equation*} \int\dfrac{1}{x^2+b^2}\, dx=\frac{1}{b}\arctan\lrpar{\dfrac{x}{b}}+C \end{equation*}

    Fact 7.4.3. Partial Fraction Decomposition.

    Let \(\dfrac{p(x)}{q(x)}\) be a rational function, where the degree of \(p\) is less than the degree of \(q\text{.}\)
    1. Linear Terms: Let \((x-a)^n\) divide \(q(x)\text{.}\) Then the decomposition of \(\frac{p(x)}{q(x)}\) will contain the terms
      \begin{equation*} \frac{A_1}{(x-a)} + \frac{A_2}{(x-a)^2} + \cdots +\frac{A_n}{(x-a)^n}\text{.} \end{equation*}
    2. Quadratic Terms: Let \((x^2+bx+c)^n\) divide \(q(x)\text{,}\) where \(x^2+bx+c\) is irreducible. Then the decomposition of \(\dfrac{p(x)}{q(x)}\) will contain the terms
      \begin{equation*} \frac{B_1x+C_1}{x^2+bx+c}+\frac{B_2x+C_2}{(x^2+bx+c)^2}+\cdots+\frac{B_nx+C_n}{(x^2+bx+c)^n}\text{.} \end{equation*}

    Example 7.4.4.

    Following is an example of a rather involved partial fraction decomposition.
    \begin{align*} &\frac{7 \, x^{6} - 4 \, x^{5} + 41 \, x^{4} - 20 \, x^{3} + 24 \, x^{2} + 11 \, x + 16}{x(x-1)^2(x^2+4)^2}\\ =& \frac{A}{x}+\frac{B}{x-1}+\frac{C}{(x-1)^2}+\frac{Dx+E}{x^2+4}+\frac{Fx+G}{(x^2+4)^2} \end{align*}
    Using some algebra, it’s possible to find values for \(A\) through \(G\) to determine
    \begin{align*} &\frac{7 \, x^{6} - 4 \, x^{5} + 41 \, x^{4} - 20 \, x^{3} + 24 \, x^{2} + 11 \, x + 16}{x(x-1)^2(x^2+4)^2}\\ =& \frac{1}{x}+\frac{2}{x-1}+\frac{3}{(x-1)^2}+\frac{4x+5}{x^2+4}+\frac{6x+7}{(x^2+4)^2}\text{.} \end{align*}

    Activity 7.4.6.

    Which of the following is the form of the partial fraction decomposition of \(\displaystyle\frac{x^3-7x^2-7x+15}{x^3(x+5)}\text{?}\)
    1. \(\displaystyle \frac{A}{x}+\frac{B}{x+5}\)
    2. \(\displaystyle \frac{A}{x^3}+\frac{B}{x+5}\)
    3. \(\displaystyle \frac{A}{x}+\frac{B}{x^2}+ \frac{C}{x^3}+\frac{D}{x+5}\)
    4. \(\displaystyle \frac{A}{x}+\frac{B}{x^2}+ \frac{C}{x^3}+\frac{Dx+E}{x+5}\)

    Activity 7.4.7.

    Which of the following is the form of the partial fraction decomposition of \(\displaystyle\frac{x^2+1}{(x-3)^2(x^2+4)^2}\text{?}\)
    1. \(\displaystyle \frac{A}{x-3}+\frac{B}{(x-3)^2}+\frac{C}{x^2+4}+\frac{D}{(x^2+4)^2}\)
    2. \(\displaystyle \frac{A}{x-3}+\frac{B}{(x-3)^2}+\frac{Cx+D}{(x^2+4)^2}\)
    3. \(\displaystyle \frac{A}{x-3}+\frac{B}{(x-3)^2}+\frac{C}{x^2+4}+\frac{Dx+E}{(x^2+4)^2}\)
    4. \(\displaystyle \frac{A}{x-3}+\frac{B}{(x-3)^2}+\frac{Cx+D}{x^2+4}+\frac{Ex+F}{(x^2+4)^2}\)

    Activity 7.4.8.

    Consider that the partial decomposition of \(\displaystyle \frac{x^2+5x+3}{(x+1)^2x}\) is
    \begin{equation*} \displaystyle \frac{x^2+5x+3}{(x+1)^2x}=\frac{A}{x+1}+\frac{B}{(x+1)^2}+\frac{C}{x}. \end{equation*}
    What equality do we obtain if we multiply both sides of the above equation by \((x+1)^2x\text{?}\)
    1. \(\displaystyle x^2+5x+3=Ax(x+1)+Bx+C(x+1)^2\)
    2. \(\displaystyle x^2+5x+3=A(x+1)+B(x+1)^2+Cx\)
    3. \(\displaystyle x^2+5x+3=Ax(x+1)+Bx+C(x+1)\)
    4. \(\displaystyle x^2+5x+3=Ax(x+1)+Bx^2+C(x+1)^2\)

    Activity 7.4.9.

    Use your choice in Activity 7.4.8 (which must hold for any \(x\) value) to answer the following.
    (a)
    By substituting \(x=0\) into the equation, we may find:
    1. \(\displaystyle A=1\)
    2. \(\displaystyle B=-2\)
    3. \(\displaystyle C=3\)
    (b)
    By substituting \(x=-1\) into the equation, we may find:
    1. \(\displaystyle A=-4\)
    2. \(\displaystyle B=1\)
    3. \(\displaystyle C=5\)

    Activity 7.4.10.

    Using the results of Activity 7.4.9, show how to rewrite our choice from Activity 7.4.8
    \begin{equation*} \unknown x^2+\unknown x=Ax^2+Ax\text{.} \end{equation*}
    What value of \(A\) satisfies this equation?
    1. \(\displaystyle -2\)
    2. \(\displaystyle 3\)
    3. \(\displaystyle 4\)
    4. \(\displaystyle -5\)

    Activity 7.4.11.

    By using the form of the decomposition \(\displaystyle \frac{x^2+5x+3}{(x+1)^2x}=\frac{A}{x+1}+\frac{B}{(x+1)^2}+\frac{C}{x}\) and the coefficients found in Activity 7.4.9 and Activity 7.4.10, evaluate \(\displaystyle \int \frac{x^2+5x+3}{(x+1)^2x} dx\text{.}\)

    Activity 7.4.12.

    Given that \(\displaystyle\frac{x^3-7x^2-7x+15}{x^3(x+5)}=\frac{A}{x}+\frac{B}{x^2}+ \frac{C}{x^3}+\frac{D}{x+5}\) do the following to find \(A, B, C\text{,}\) and \(D\text{.}\)
    (a)
    Eliminate the fractions to obtain
    \begin{equation*} x^3-7x^2-7x+15=A(\unknown)(\unknown)+B(\unknown)(\unknown)+C(\unknown)+D(\unknown)\text{.} \end{equation*}
    (b)
    Plug in an \(x\) value that lets you find the value of \(C\text{.}\)
    (c)
    Plug in an \(x\) value that lets you find the value of \(D\text{.}\)
    (d)
    Use other algebra techniques to find the values of \(A\) and \(B\text{.}\)
