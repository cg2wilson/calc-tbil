    Section 0.1 Functions

    Subsection 0.1.1 Basics

    Definition 0.1.1. Relation/Domain/Range.

    A relation is a collection of ordered pairs of the form \((x,y)\text{.}\) The collection of first components in a relation is called the domain of the relation. The collection of second components in a relation is called the range of the relation.

    Definition 0.1.2. Input/Output.

    In an ordered pair \((x,y)\text{,}\) the element \(x\) is called an input and the element \(y\) is called an output.

    Definition 0.1.3. Function.

    A function \(f\) is a rule which assigns to each input \(x\) in the domain a single output \(f(x)\) in the range.

    Example 0.1.4.

    The relation \(f\) is copied below. Is \(f\) a function? Why or why not?
    \begin{equation*} \lrbrace{(-2,1), (1,5), (0,-3), (-2,3), (10,1), (3,3)} \end{equation*}
    Solution.
    No, because the input \(-2\) is assigned two outputs: \(1\) and \(3\text{.}\)

    Definition 0.1.5. Graph.

    The graph of a function is the collection of all coordinate pairs \((x,y)\text{,}\) plotted on a coordinate system.

    Theorem 0.1.6. Vertical Line Test.

    A relation is a function if and only if any vertical intersects the graph of the relation in at most one point.

    Example 0.1.7.

    Are both of these graphs functions? Why or why not?
    Solution.
    The graph in Figure 0.1.8 is a function, because every input has only one output; however, the graph in Figure 0.1.9 is not a function because all inputs in \([-2,2]\)

    Example 0.1.10.

    Below are numerical expressions for \(h\) and \(k\text{.}\) Is \(h\) a function? What about \(k\text{?}\)
    \(x\) \(0\) \(1\) \(1\) \(2\) \(5\) \(6\)
    \(h(x)\) \(0\) \(1\) \(2\) \(3\) \(4\) \(5\)
    \(t\) \(0\) \(1\) \(1\) \(2\) \(5\) \(6\)
    \(k(t)\) \(0\) \(1\) \(1\) \(3\) \(4\) \(5\)
    Solution.
    \(h\) is not a function because the input \(1\) has two outputs: \(1\) and \(2\text{.}\) However, \(k\) is a function because every input has only one output.
    Even though the input \(1\) appears twice, it has the same output. So in a sense, we can consider this a “duplicate” entry.

    Subsection 0.1.2 Using Functions

    Remark 0.1.11. Function Notation: Functions of a Single Variable.

    The notation \(f(x)\) has two pieces: an input and a rule. We are applying the rule \(f\) to the input \(x\text{.}\) The result is an output, usually denoted \(y\text{.}\) So we can write
    \begin{equation*} y = f(x) \end{equation*}
    as a way to say “the output \(y\) is the result of applying the rule \(f\) to the input \(x\)”. This should be read as “\(y\) equals \(f\) of \(x\)”

    Remark 0.1.12. Evaluating Functions.

    To evaluate a function \(f(x)\text{,}\) replace all instances of the variable \(x\) with the input shown.

    Example 0.1.13.

    Let \(f(x) = x^2 + 5\text{.}\) Evaluate the following:
    1. \(\displaystyle f(1)\)
    2. \(\displaystyle f(0)\)
    3. \(\displaystyle f(-2)\)
    4. \(\displaystyle f(a)\)
    5. \(\displaystyle f(x + h)\)
    Solution.
    1. \(\displaystyle f(1)=6\)
    2. \(\displaystyle f(0)=5\)
    3. \(\displaystyle f(-2)=9\)
    4. \(\displaystyle f(a)=a^2+5\)
    5. \(\displaystyle f(x + h)=x^2+2xh+h^2+5\)

    Example 0.1.14.

    Consider the following function rules: \(f(x) = x-1, g(x) = -x^2, h(x) = 2^x\text{.}\) Write the function notation for the descriptions given, then compute.
    1. \(f\) applied to the input 5
    2. \(g\) applied to the input \(a + 1\)
    3. \(h\) applied to the input \(3 + r\)
    Solution.
    1. \(\displaystyle f(5) = 4\)
    2. \(\displaystyle g(a+1)=-a^2-2ah-h^2\)
    3. \(\displaystyle h(3+r) = 2^{3+r} = 8\cdot 2^r\)

    Subsection 0.1.3 Combining Functions

    Function Combinations.

    Name Notation Translation Note
    Sum \((f+g)(x)\) \(f(x) + g(x)\)
    Difference \((f-g)(x)\) \(f(x) - g(x)\)
    Product \((f\cdot g)(x)\) \(f(x)\cdot g(x)\) \((f\cdot g)\) can be written as \((fg)\) as well
    Quotient \(\lrpar{\dfrac{f}{g}}(x)\) \(\dfrac{f(x)}{g(x)}\) \(g(x)\) cannot be zero

    Example 0.1.15.

    Let \(f(x) = 3x-1\) and \(g(x) = x^2+x+1\text{.}\) Write and simplify the following.
    1. \(\displaystyle (f+g)(x)\)
    2. \(\displaystyle (f-g)(x)\)
    3. \(\displaystyle (fg)(x)\)
    4. \(\displaystyle \lrpar{\dfrac{f}{g}}(x)\)
    Solution.
    1. \(\displaystyle (f+g)(x) = x^2 + 4x\)
    2. \(\displaystyle (f-g)(x) = -x^2 + 2x - 2\)
    3. \(\displaystyle (fg)(x) = 3x^3+2x^2+x-1\)
    4. \(\displaystyle \lrpar{\dfrac{f}{g}}(x) = \dfrac{3x-1}{x^2+x-1}\)

    Remark 0.1.16.

    Functions can also be combined using composition: using one function as an input to another. The notation is given by
    \begin{equation*} (f\circ g)(x) = f(g(x)) \end{equation*}

    Example 0.1.17.

    Let \(f(x) = 3x-1\) and \(g(x) = x^2+x+1\text{.}\) Write and simplify the following.
    1. \(\displaystyle (f\circ g)(x)\)
    2. \(\displaystyle (g\circ f)(x)\)
    3. \(\displaystyle g(-4x)\)
    4. \(\displaystyle f(g(-x))\)
    Solution.
    1. \(\displaystyle (f\circ g)(x)= 3x^2+3x+2\)
    2. \(\displaystyle (g\circ f)(x)=9x^2-3x+1\)
    3. \(\displaystyle g(-4x)=-12x-1\)
    4. \(\displaystyle f(g(-x))=3x^2-3x+2\)

    Remark 0.1.18.

    Given two functions \(f(x)\) and \(g(x)\text{,}\) the composition \((f\circ g)(x)\) has the domain
    \begin{equation*} \text{Dom}\,(f\circ g) = \lrbrace{x\mid x\in \text{Dom}\,(g)\text{ and }g(x)\in\text{Dom}\, (f)} \end{equation*}
    and the range
    \begin{equation*} \text{Range}\, (f\circ g) = \lrbrace{y=f(x)\mid x\in \text{Range}\, (g)} \end{equation*}

    Example 0.1.19.

    Consider the functions \(f\) and \(g\) given below.
    \(x\) \(-3\) \(-2\) \(-1\) \(0\) \(1\) \(2\) \(3\) \(4\)
    \(f(x)\) \(0\) \(4\) \(2\) \(4\) \(-2\) \(0\) \(-2\) \(4\)
    \(x\) \(-4\) \(-2\) \(0\) \(2\) \(4\)
    \(g(x)\) \(1\) \(0\) \(3\) \(0\) \(5\)
    Find the following.
    1. \(\displaystyle (g\circ f)(3)\)
    2. The domain and range of \((g\circ f)(x)\)
    3. The domain and range of \((f\circ g)(x)\)
    Solution.
    1. \(\displaystyle (g\circ f)(3)=0\)
    2. \(\text{Dom}\,(g\circ f) = \lrbrace{-2,0,2,4}\) and \(\text{Range}\, (g\circ f) = \lrbrace{0,3,5}\)
    3. \(\text{Dom}\,(f\circ g) = \lrbrace{-4,-2,0,2}\) and \(\text{Range}\, (f\circ g) = \lrbrace{-2,4}\)

    Example 0.1.20.

    For each function below, completely decompose the function into smaller ones and find the domain of the composite.
    1. \(\displaystyle f(x) = \dfrac{1}{x+2}\)
    2. \(\displaystyle q(x) = (2x+1)^5\)
    3. \(\displaystyle s(h) = \sin \lrpar{5h^2 + \dfrac{1}{h}}\)
    4. \(\displaystyle y(r) = \dfrac{5.317}{(2r^5+1.7)^2}\)
    Solution.
    Decompositions can vary; the answers shown here are suggested.
    1. \(f(x) = a(b(x))\) where \(a(x) = \dfrac{1}{x}\) and \(b(x) = x+2\)
    2. \(q(x) = a(b(x))\) where \(a(x) = x^5\) and \(b(x) = 2x+1\)
    3. \(s(h) = a(b(h))\) where \(a(h) = \sin (h)\) and \(b(h) = 5h^2 + \dfrac{1}{h}\)
    4. \(y(r) = a(b(r))\) where \(a(r) = \dfrac{5.317}{r^2}\) and \(b(r) = 2r^5 + 1.7\)

    Example 0.1.21.

    Let \(k(x) = \sec (x^2)\tan(x^2)\text{.}\) Find \(f,g\) such that \(k(x) = f(g(x))\)
    Solution.
    \(f(x) = \sec (x)\tan (x)\) and \(g(x) = x^2\)

    Example 0.1.22.

    Let \(f(x) = \cos^2 (x^2+9)\text{.}\) Find functions \(a,b,c\) such that \(f(x) = (a\circ b\circ c)(x)\)
    Solution.
    \(a(x) = x^2\text{,}\) \(b(x) = \cos (x)\text{,}\) \(c(x) = x^2 + 9\)

    Subsection 0.1.4 Symmetry

    Definition 0.1.23.

    A function \(f(x)\) is said to be even if it has the property that
    \begin{equation*} f(-x) = f(x) \end{equation*}
    for all \(x\) in its domain.
    A function \(g(x)\) is said to be odd if it has the property that
    \begin{equation*} g(-x) = -g(x) \end{equation*}
    for all \(x\) in its domain.

    Example 0.1.24.

    Determine if the following functions are even, odd, or neither.
    1. \(\displaystyle f(x) = x^7 + x^5 -x\)
    2. \(\displaystyle g(x) = 3-x^2\)
    3. \(\displaystyle h(x) = x^2-x^3\)
    Solution.
    1. Odd
    2. Even
    3. Neither

    Subsection 0.1.5 Transformations of Functions

    Vertical and Horizontal Shifts.

    Let \(c\gt 0\text{.}\) To obtain the graph of
    • \(y = f(x) + c\text{,}\) shift the graph of \(y=f(x)\) up by \(c\) units
    • \(y = f(x) - c\text{,}\) shift the graph of \(y=f(x)\) down by \(c\) units
    • \(y = f(x+c)\text{,}\) shift the graph of \(y=f(x)\) left by \(c\) units
    • \(y = f(x-c)\text{,}\) shift the graph of \(y=f(x)\) right by \(c\) units

    Vertical and Horizontal Stretching/Reflecting.

    Let \(c \gt 1\text{.}\) To obtain the graph of
    • \(y = c\cdot f(x)\text{,}\) stretch the graph of \(y=f(x)\) vertically by a factor of \(c\)
    • \(y=\lrpar{\dfrac{1}{c}}\cdot f(x)\text{,}\) shrink the graph of \(y=f(x)\) vertically by a factor of \(c\)
    • \(y= f(c\cdot x)\text{,}\) shrink the graph of \(y=f(x)\) horizontally by a factor of \(c\)
    • \(y = f\lrpar{\dfrac{1}{c}\cdot x}\text{,}\) stretch the graph of \(y=f(x)\) horizontally by a factor of \(c\)
    • \(y=-f(x)\text{,}\) reflect the graph of \(y=f(x)\) about the \(x-\)axis
    • \(y=f(-x)\text{,}\) reflect the graph of \(y=f(x)\) about the \(y-\)axis

    Example 0.1.25.

    If \(g(1) = 3\text{,}\) what point must be on the graph of \(h(t) = -2g(t-1)+6\text{?}\)
    Solution.
    \(\lrpar{2,0}\)
