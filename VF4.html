    Section 12.4 VF4: Fundamental Theorem of Line Integrals

    Objectives

    VF4
    I can use the Fundamental Theorem of Line Integrals to evaluate a line integral in a vector field, and I can find the potential function for a conservative vector field.

    Pre-Class Preparation.

    Read the following sections of the text:
    • Section 6.3: Conservative Vector Fields
      Read all sections

    Subsection 12.4.1 In-Class Activities

    Definition 12.4.1. Closed Curve/Simple Curve.

    A curve \(C\) is said to be closed if there is a parameterization of \(C\) given by \(\textbf{r}(t)\) for \(a\leq t\leq b\) such that \(\textbf{r}(a) = \textbf{r}(b)\) and the parameterization traverses the curve only once.
    A curve is said to be simple if it does not intersect itself, except possibly \(t = a = b\)

    Activity 12.4.1.

    The curve traced by the parameterization \(\textbf{r}(t) = \lrangle{2\cos t, 3\sin t}\text{,}\) \(0\leq 6\pi\) is shown below.
    1. The curve is simple and closed
    2. The curve is neither simple nor closed
    3. The curve is simple but not closed
    4. The curve is closed but not simple

    Definition 12.4.2. Connected Region/Simply Connected Region.

    A region \(D\) is said to be connected if for any two points in \(D\text{,}\) there exists a path between them which lies entirely in \(D\text{.}\)
    \(D\) is further said to be simply connected if all it can be compressed to a single point. In particular, \(D\) cannot have any holes and must be connected.

    Theorem 12.4.3. Fundamental Theorem of Line Integrals.

    Let \(C\) be a piecewise-smooth curve with parameterization \(\textbf{r}(t)\text{,}\) \(a\leq t\leq b\text{.}\) Let \(f\) be a function of two or three variables with first-order partial derivatives that exist and are continuous on \(C\text{.}\) Then,
    \begin{equation*} \int_C \nabla f\cdot d\textbf{r} = f(\textbf{r}(b)) - f(\textbf{r}(a)) \end{equation*}

    Activity 12.4.2.

    Consider the vector field
    \begin{equation*} \textbf{F} = \lrangle{2xy-2y+(y+1)^2,(x-1)^2 + 2yx+2x} \end{equation*}
    (a)
    Confirm that \(f(x,y) = (x-1)^2y + (y+1)^2x\) is a potential function for \(\textbf{F}\)
    (b)
    If \(C\) is the lower half of the unit circle, oriented counterclockwise, compute \(\ds \int_C \textbf{F}\cdot d\textbf{r}\)
    (c)
    Now compute \(\ds \int_C \textbf{F}\cdot d\textbf{r}\) if \(C\) is the straight-line path \((-1,0)\) to \((1,0)\text{.}\)

    Definition 12.4.4. Path Independence.

    Let \(\textbf{F}\) be a vector field with domain \(D\text{.}\) \(\textbf{F}\) is said to the path independent if \(\ds \int_{C_1}\textbf{F}\cdot d\textbf{r} = \int_{C_2} \textbf{F}\cdot d\textbf{r}\) for any paths \(C_1,C_2\) in \(D\) with the same initial and terminal points.

    Fact 12.4.5.

    If \(\textbf{F}\) is a conservative vector field, then \(\textbf{F}\) is path independent

    Fact 12.4.6.

    Let \(\textbf{F}\) be a continuous vector field. Then, \(\textbf{F}\) has path independent line integrals if and only if \(\oint_C \textbf{F}\cdot d\textbf{r} = 0\) for all piecewise smooth, simple, closed curves in the domain of \(\textbf{F}\text{.}\)

    Activity 12.4.3.

    True or false: the vector field \(\lrangle{x^2y,y+5}\) is conservative
    1. True
    2. False

    Activity 12.4.4.

    Compute the circulation for the vector field
    \begin{equation*} \textbf{F} = \lrangle{2xy+\cos(2y),x^2-2x\sin(2y)} \end{equation*}
    along the ellipse \(\dfrac{x^2}{4} + \dfrac{y^2}{9} =1 \)

    Activity 12.4.5.

    Find a potential function for \(\textbf{F}(x,y) = \lrangle{2xy^3,3x^2y^2+\cos(y)}\text{.}\)

    Activity 12.4.6.

    Determine if the vector field
    \begin{equation*} \textbf{F} = \lrangle{2xy+\cos(2y),x^2-2x\sin(2y)} \end{equation*}
    is conservative; if it is, find its potential function.

    Activity 12.4.7.

    Find a potential function for \(\textbf{F}(x,y) = \lrangle{e^xy^3+y,3e^xy^2+x}\)

    Activity 12.4.8.

    Compute the line integral \(\ds \int_C \textbf{F}\cdot d\textbf{r}\text{,}\) where
    \begin{equation*} \textbf{F}(x,y,z) = \lrangle{2xe^yz+e^xz,x^2e^yz,x^2e^y+e^x} \end{equation*}
    and \(C\) is any smooth curve that goes from the origin to \((1,1,1)\)

    Activity 12.4.9.

    This activity will prove the Law of Conservation of Energy. Let \(\textbf{F}\) be a continuous, conservative force field which moves an object along a path \(C\) given by \(\textbf{r}(t)\text{,}\) \(a\leq t\leq b\text{,}\) where \(\textbf{r}(a) = A\) is the initial point and \(\textbf{r}(b) = B\) is the terminal point.
    (a)
    Use a line integral to describe the work done in moving the particle along \(C\text{;}\) use Newton’s Second Law to simplify the expression.
    (b)
    Use the fact that \(\dfrac{d}{dt}\left[\textbf{r}'(t)\cdot \textbf{r}'(t)\right] = 2\textbf{r}''(t)\cdot \textbf{r}'(t)\) to rewrite the integral.
    (c)
    Use the Fundamental Theorem of Calculus to show that the work is given by
    \begin{equation*} W = \dfrac{1}{2}m |\textbf{v}(b)|^2 - \dfrac{1}{2}m|\textbf{v}(a)|^2 \end{equation*}
    where \(\textbf{r}' = \textbf{v}\)
    (d)
    What is the relationship between the vector field and the potential energy of the object?
    (e)
    Use the relationship and the Fundamental Theorem of Line Integrals to show that work is the net change in potential energy.
