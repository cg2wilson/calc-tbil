    Section 2.3 DE3: Slope Graphs & Differentiability

    Objectives

    DE3
    I can sketch the derivative graph of a function and determine where it is differentiable (and classify the non-differentiability type) using numerical, algebraic, or graphical representations.

    Activity 2.3.5.

    Consider the function \(f(x) = \dfrac{x^2+11x+30}{|x+6|}\text{.}\)
    (a)
    Remember that \(|x| = \begin{cases} -x & x\lt 0 \\ x & x\geq 0\end{cases}\text{.}\) Use this fact to compute \(\ds \lim_{x\to 6^+} f(x)\text{.}\)
    (b)
    Now compute \(\ds \lim_{x\to 6^-} f(x)\)
    (c)
    True or false: \(\ds \lim_{x\to 6} f(x) \) exists. Be prepared to defend your answer with a reason why if it exists, and why not if it doesn’t.
    1. True
    2. False

    Activity 2.3.6.

    Now consider the function \(g(x) = |x|\text{.}\) Define the left-hand derivative to be the expression
    \begin{equation*} g'_- (x) = \lim_{h\to 0^-} \dfrac{g(x+h) - g(x)}{h} \end{equation*}
    and the right-hand derivative to be the expression
    \begin{equation*} g'_+ (x) = \lim_{h\to 0^+} \dfrac{g(x+h)-g(x)}{h} \end{equation*}
    (a)
    Which is the appropriate value of \(g'_-(0)\text{?}\)
    1. \(\displaystyle g'_-(0) = 0\)
    2. \(\displaystyle g'_-(0) = 1\)
    3. \(\displaystyle g'_-(0) = -1\)
    4. \(g'_-(0)\) DNE
    (b)
    Which is the appropriate value of \(g'_+(0)\text{?}\)
    1. \(\displaystyle g'_+(0) = 0\)
    2. \(\displaystyle g'_+(0) = 1\)
    3. \(\displaystyle g'_+(0) = -1\)
    4. \(g'_+(0)\) DNE
    (c)
    Your answers to (a) and (b) should lead to you conclude what about the value of
    \begin{equation*} \lim_{h\to 0} \dfrac{g(x+h)-g(x)}{h} \end{equation*}
    1. The limit exists and therefore \(g'(0) = 0\)
    2. The limit exists and therefore \(g'(0) = 1\)
    3. The limit exists and therefore \(g'(0) = -1\)
    4. The limit does not exist and therefore \(g'(0)\) does not exist
    (d)
    Recall that the expression \(\dfrac{f(x+h)-f(x)}{h}\) computes the slope of a function \(f(x)\) between the points \((x,f(x))\) and \((x+h,f(x+h))\text{.}\) Based on this, complete the statement: “If the slopes of the tangent lines to a function at a point are not the same from the left and from the right, then...”

    Definition 2.3.9.

    A function \(f(x)\) is said to be differentiable at the point \(x=a\) if the limit
    \begin{equation*} \lim_{h\to 0} \dfrac{f(a+h)-f(a)}{h} \end{equation*}
    exists

    Activity 2.3.7.

    Using Definition 2.3.9, write an explicit, mathematical statement for why \(f(x) = |x|\) is not differentiable at \(x=0\text{.}\)
