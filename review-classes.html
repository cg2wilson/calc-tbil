    Section 0.2 Classes of Functions

    Subsection 0.2.1 Linear Functions

    Definition 0.2.1. Linear Function.

    A linear function takes the form
    \begin{equation*} f(x) = mx + b \end{equation*}
    where \(m\) is the slope of the line and \((0,b)\) is the location of the \(y-\)intercept.

    Definition 0.2.2. Slope.

    The slope of a line is the ratio of the change in output value when the input changes by a specific amount.
    Given points \((x_1,y_1)\) and \((x_2,y_2)\) on a line, the slope of the line is found by
    \begin{equation*} m = \dfrac{y_2 - y_1}{x_2-x_1} = \dfrac{\Delta y}{\Delta x} \end{equation*}
    If we write \(y_1 = f(x_1)\) and \(y_2 = f(x_2)\text{,}\) then we can rewrite slope as
    \begin{equation*} m = \dfrac{f(x_2)-f(x_1)}{x_2-x_1} = \dfrac{\Delta y}{\Delta x} \end{equation*}

    Three Forms of a Line.

    A line is in slope-intercept form if it is written as
    \begin{equation*} y = mx + b \end{equation*}
    where \(m\) is the slope of the line and \((0,b)\) is the location of the \(y-\)intercept.
    It is in point-slope form if it is written as
    \begin{equation*} y-y_1 = m(x-x_1) \end{equation*}
    where \(m\) is the slope of the line and the line goes through the point \((x_1,y_1)\text{.}\)
    The line is in standard form if it is written as
    \begin{equation*} ax + by = c \end{equation*}
    where \(a,b,c\) are real numbers such that \(a,b\) are not both zero.

    Four Types of Lines.

    Consider a line of the form \(ax + by = c\text{.}\)
    • If \(a,b\neq 0\text{,}\) then the line has slope \(m = -\dfrac{a}{b}\) and passes through the point \(\lrpar{0,\dfrac{c}{b}}\text{.}\) In particular, the line is neither horizontal nor vertical.
      If \(m \lt 0\text{,}\) then the line has negative slope; if \(m \gt 0\text{,}\) then the line has positive slope.
    • If \(a = 0\text{,}\) then the line has slope \(m = 0\) and passes through the point \(\lrpar{0,\dfrac{c}{b}}\text{.}\) In particular, the line is horizontal with equation
      \begin{equation*} y = \dfrac{c}{b} \end{equation*}
    • If \(b=0\text{,}\) then the line has undefined slope and passes through the point \(\lrpar{\dfrac{c}{a},0}\text{.}\) In particular, it is vertical with equation
      \begin{equation*} x = \dfrac{c}{a} \end{equation*}

    Subsection 0.2.2 Polynomials

    Definition 0.2.3.

    A function \(P\) is called a polynomial if it takes the form
    \begin{equation*} P(x) = a_nx^n + a_{n-1}x^{n-1} + \cdots + a_2x^2+a_1x+a_0 \end{equation*}
    where \(n\) is a non-negative integer and the numbers \(a_i\) are called the coefficients of the polynomial. The degree is the power of the leading term (\(n\)).

    Domain and Range of Polynomials.

    Let \(a_nx^n + a_{n-1}x^{n-1}+\cdots a_1x + a_0\) be a polynomial, and let \(k\) be some real number
    Type Domain Range
    Even Degree, \(a_n \gt 0\) \((-\infty,\infty)\) \((k,\infty)\)
    Even Degree, \(a_n \lt 0\) \((-\infty,\infty)\) \((-\infty,k)\)
    Odd Degree, \(a_n \gt 0\) \((-\infty,\infty)\) \((-\infty,\infty)\)
    Odd Degree, \(a_n \lt 0\) \((-\infty,\infty)\) \((-\infty,\infty)\)

    Remark 0.2.4.

    Some common polynomials are:
    • Linear: These are described above
    • Quadratic: They take the form \(f(x) = ax^2 + bx + c\)
    • Cubic: These take the form \(f(x) = ax^3 + bx^2 + cx + d\)

    Zeros of a Polynomial.

    The zeros of any polynomial \(f(x)\) are found by solving the equation
    \begin{equation*} f(x) = 0 \end{equation*}
    In paticular, if \(f(x)\) has zeros \(x_1,...,x_k\text{,}\) then \(f(x)\) can be written as
    \begin{equation*} f(x) = (x-x_1)(x-x_2)\cdots (x-x_k) \end{equation*}
    A zero is said to have multiplicity \(m\) if the factor \(x-x_j\) occurs \(m\) times.

    Properties and Zeros of a Quadratic.

    A quadratic function \(f(x) = ax^2 + bx + c\) has
    • Two real solutions if \(b^2-4ac \gt 0\)
    • One real solution if \(b^2-4ac = 0\)
    • No real solutions if \(b^2 -4ac \lt 0\)
    The vertex of a quadratic function is located at the point
    \begin{equation*} \lrpar{-\dfrac{b}{2a},f\lrpar{-\dfrac{b}{2a}}} \end{equation*}
    The zeros, if they exist, are located at
    \begin{equation*} x = -\dfrac{b}{2a} - \dfrac{\sqrt{b^2-4ac}}{2a} \end{equation*}
    and
    \begin{equation*} x = -\dfrac{b}{2a} + \dfrac{\sqrt{b^2-4ac}}{2a} \end{equation*}
    i.e., the zeros are always \(\dfrac{\sqrt{b^2-4ac}}{2a}\) units away from the vertex.

    Quadratic Formula.

    For a quadratic function of the form \(f(x) = ax^2 + bx + c\text{,}\) the solutions are given by
    \begin{equation*} x = \dfrac{-b\pm \sqrt{b^2-4ac}}{2a} \end{equation*}
    The term \(b^2-4ac\) is called the discriminant of the quadratic.

    Example 0.2.5.

    Consider the quadratic function \(f(x) = 2x^2-9x-5\text{.}\)
    1. Use the quadratic formula to identify the zeros of the function.
    2. Find the zeros again, but this time by factoring.
    3. What is the range of \(f(x)\text{?}\)
    Solution.
    1. Identifying \(a=2,b=-9,c=-5\text{,}\) we have
      \begin{equation*} x = \dfrac{9\pm \sqrt{(-9)^2-4(2)(-5)}}{2(2)} \end{equation*}
      which yields the solutions \(x = -\dfrac{1}{2}\) and \(x=5\)
    2. Using any preferred method, we see that
      \begin{equation*} 2x^2-9x-5 = (2x+1)(x-5) \end{equation*}
      which gives the same zeros as in (a).
    3. Since \(a \gt 0\text{,}\) and the vertex is at
      \begin{equation*} \lrpar{\dfrac{9}{4},-\dfrac{121}{8}} \end{equation*}
      the range is
      \begin{equation*} \lrpar{-\dfrac{121}{8},\infty} \end{equation*}

    Subsection 0.2.3 Rational Functions

    Definition 0.2.6. Rational Functions.

    A rational function is one which takes the form
    \begin{equation*} f(x) = \dfrac{P(x)}{Q(x)} \end{equation*}
    where \(P(x)\) and \(Q(x)\) are both polynomials

    Domain of Rational Functions.

    The domain of a rational function \(f(x) = \dfrac{P(x)}{Q(x)}\) is the set
    \begin{equation*} \lrbrace{x\mid Q(x)\neq 0} \end{equation*}

    Subsection 0.2.4 Power Functions

    Definition 0.2.7. Power Functions.

    A power function is any function of the form
    \begin{equation*} f(x) = x^a \end{equation*}
    where \(a\) is any real number.

    Remark 0.2.8.

    There are two power functions which see special use:
    • If \(a = \dfrac{1}{n}\text{,}\) where \(n\geq 2\) is an integer, then we call the function
      \begin{equation*} f(x) = x^a = x^{1/n} \end{equation*}
      a root function, since the rules of exponents say that
      \begin{equation*} x^{1/n} = \sqrt[n]{x} \end{equation*}
      If \(n=2\text{,}\) then we have a square root function; if \(n=3\text{,}\) we have a cube root function.
    • If \(a=-1\text{,}\) then the function
      \begin{equation*} f(x) = x^{-1}=\dfrac{1}{x} \end{equation*}
      is called the reciprocal function.
      The reciprocal function can also be considered a rational function, with \(P(x) = 1\) and \(Q(x) = x\text{.}\)

    Subsection 0.2.5 Algebraic & Transcendental Functions

    Definition 0.2.9. Algebraic Function.

    The function \(f(x)\) is called algebraic if it can be constructed using algebraic operations such as addition, subtraction, multiplication, division, and taking roots, starting with polynomials.

    Example 0.2.10.

    Find the domain of the function \(f(x) = \dfrac{9x^2-1}{\sqrt{4x+5}}\)
    Solution.
    \(\lrpar{-\dfrac{5}{4},\infty}\)

    Definition 0.2.11. Exponential & Logarithmic Function.

    An exponential function is a function of the form
    \begin{equation*} f(x) = b^x \end{equation*}
    where the base \(b\) is a positive constant.
    A logarithmic function is a function of the form
    \begin{equation*} g(x) = \log_b(x) \end{equation*}
    where the base \(b\) is a positive constant.

    Domain and Range of Transcendental Functions.

    Function Domain Range Asymptote
    \(f(x)=b^x\) \((-\infty,\infty)\) \((0,\infty)\) \(y = 0\)
    \(f(x)=\log_b(x)\) \((0,\infty)\) \((-\infty,\infty)\) \(x=0\)

    Example 0.2.12.

    Find the domain of the function \(k(r) = \log_5(r^2-2r+3)\)
    Solution.
    \((-\infty,-1)\cup (3,\infty)\)

    Subsection 0.2.6 Piecewise Functions

    Definition 0.2.13. Piecewise Functions.

    A piecewise function is a function defined by different formulas in different parts of its domain.

    Remark 0.2.14.

    The most common piecewise function you will encounter in Calculus is the absolute value function, which is defined by
    \begin{equation*} |x| = \begin{cases} -x \amp x\lt 0 \\ x \amp x\geq 0 \end{cases} \end{equation*}
