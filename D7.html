    Section 2.7 D7: Derivative Rules

    Objectives

    D7
    I can compute derivatives using one or more of the following rules or techniques: product, quotient, inverse.

    Pre-Class Preparation.

    Read the following sections of the text:
    • Section 3.3: Differentiation Rules
      Read the "The Product Rule", "The Quotient Rule", and "Combining Differentiation Rules" sections
    • Section 3.7: Derivatives of Inverse Functions
      Read all sections
    It will also likely be helpful to review the section on inverses: Section0.4

    Subsection 2.7.1 In-Class Activities

    Activity 2.7.1.

    Let \(f(t) = 8t^5 -6\sqrt{t}\) and \(g(t) = 2t\text{.}\)
    (a)
    Compute \(f'(t)\) and \(g'(t)\)
    (b)
    (a)
    Find the product function, \(P(t) = f(t)\cdot g(t)\text{.}\)
    (b)
    Compute \(P'(t)\)
    (c)
    True or false: \(P'(t) = f'(t)\cdot g'(t)\)
    (c)
    (a)
    Find the quotient function, \(Q(t) = \dfrac{f(t)}{g(t)}\text{.}\)
    (b)
    Compute \(Q'(t)\)
    (c)
    True or false: \(Q'(t) = \dfrac{f'(t)}{g'(t)}\)

    Theorem 2.7.1. The Product Rule.

    Let \(f(x)\) and \(g(x)\) be differentiable functions. Then,
    \begin{equation*} \dfrac{d}{dx}[f(x)\cdot g(x)] = f'(x)g(x) + f(x)g'(x) \end{equation*}

    Theorem 2.7.2. The Quotient Rule.

    Let \(f(x)\) and \(g(x)\) be differentiable functions. Then,
    \begin{equation*} \dfrac{d}{dx}\left[\dfrac{f(x)}{g(x)}\right] = \dfrac{f'(x)g(x) - f(x)g'(x)}{(g(x))^2} \end{equation*}
    provided \(g(x)\neq 0\text{.}\)

    Activity 2.7.2.

    The product rule is a powerful tool, but sometimes it isnt necessary; a more elementary rule may suffice. For which of the following functions can you find the derivative without using the product rule? Select all that apply and find the derivative.
    1. \(\displaystyle f(x)=e^x \sin(x)\)
    2. \(\displaystyle f(x)=\sqrt{x}(x^3+3x-3)\)
    3. \(\displaystyle f(x)=(4)(x^5)\)
    4. \(\displaystyle f(x)=x \ln(x)\)

    Activity 2.7.3.

    Use the product rule to find the derivative of the following functions.
    (a)
    \(f(x)=(x^2+3x)\tan(x)\)
    (b)
    \(f(x)=2^x \cos(x)\)
    (c)
    \(f(x) = x^2\inv{\sin}(x)\)

    Activity 2.7.4.

    Just like with the product rule, there are times when we can find the derivative of a quotient using elementary rules rather than the quotient rule. For which of the following functions can you find the derivative without using the quotient rule? Select all that apply and find the derivative.
    1. \(\displaystyle f(x) = \dfrac{6}{x^3}\)
    2. \(\displaystyle f(x) = \dfrac{-5}{\ln(x)}\)
    3. \(\displaystyle f(x) = \dfrac{e^x}{\csc(x)}\)
    4. \(\displaystyle f(x) = \dfrac{x^3+3x}{x}\)

    Activity 2.7.5.

    Find the derivative of the following functions using the quotient rule (or, if applicable, an elementary rule).
    (a)
    \(f(x) = \dfrac{\inv{\tan}(x)}{x^3}\)
    (b)
    \(g(x) = \dfrac{x^2 + x^{5/3}}{x^4}\)
    (c)
    \(h(x) = \dfrac{5e^x - \cos(x)}{x}\)

    Activity 2.7.6.

    Use the quotient rule and appropriate trigonometric identities to prove the following:
    (a)
    \(\dfrac{d}{dx}[\tan(x)] = \sec^2(x)\)
    (b)
    \(\dfrac{d}{dx}[\cot(x)] = -\csc^2(x)\)
    (c)
    \(\dfrac{d}{dx}[\sec(x)] = \sec(x)\tan(x)\)
    (d)
    \(\dfrac{d}{dx}[\csc(x)] = -\csc(x)\cot(x)\)

    Theorem 2.7.3. Derivative of Inverse Functions (at a Point).

    Let \(f\) be one-to-one and differentiable at \(x=a\text{,}\) with inverse function \(\inv{f}\) and \(f'(\inv{f}(a)) \neq 0\text{.}\) Then, the inverse function is differentiable at \(x=a\) and
    \begin{equation*} (\inv{f})'(a) = \dfrac{1}{f'(\inv{f}(a))} \end{equation*}

    Activity 2.7.7.

    Let \(f(x) = 3x - \sin(x)\text{.}\) In order to find \((\inv{f})'(0)\text{,}\) we need to verify that we can use the formula in Theorem2.7.3.
    (a)
    One way to check if a function is one-to-one is by taking its derivative and determining if it is always positive or always negative.
    1. \(f'(x) = 3 + \cos(x)\text{,}\) and it is always negative
    2. \(f'(x) = 3x -\cos(x)\text{,}\) and switches sign
    3. \(f'(x) = 3 -\cos(x)\text{,}\) and switches sign
    4. \(f'(x) = 3-\cos(x)\text{,}\) and is always positive
    (b)
    Instead of finding the inverse function, \(\inv{f}(0)\text{,}\) we can take advantage of the fact that
    \begin{equation*} \inv{f}(0) = x \iff f(x) = 0 \end{equation*}
    What value (or values) makes this statement true?
    1. \(\displaystyle x = 0\)
    2. \(\displaystyle x = \pi\)
    3. \(x = 0\) and \(x = \pi\)
    4. \(\displaystyle x = \dfrac{\pi}{2}\)
    (c)
    Now we can compute \(f'(\inv{f}(0))\text{.}\) What is its value?
    (d)
    Now finish the formula. What is the value of \((\inv{f})'(0)\text{?}\)

    Activity 2.7.8.

    Suppose that \(\inv{f}\) is the inverse function of a differentiable function \(f\) with \(f(4) = 5\) and \(f'(4) = \dfrac{2}{3}\text{.}\) Find \((\inv{f})'(5)\)
