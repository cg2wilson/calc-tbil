    Section 10.2 PD2: Limits and Continuity

    Objectives

    PD2
    I can compute the limit of a function of multiple variables and identify where a function of two variables is discontinuous.

    Pre-Class Preparation.

    Read the following sections of the text:
    • Section 4.2: Limits and Continuity
      Read all sections

    Subsection 10.2.1 In-Class Activities

    Remark 10.2.1.

    Recall that in single-variable calculus, we define a limit as follows: The function \(f(x)\) has limit \(L\) as \(x\) approaches \(a\) if
    \begin{equation*} \lim_{x\to a^-} f(x) = \lim_{x\to a^+} f(x) = L \end{equation*}
    i.e. the output approaches \(L\) as \(x\) approaches \(a\) from both the left and right sides.

    Definition 10.2.2. Limit of a Function of Two Variables.

    Let \(f(x,y)\) be a function of two variables. We say that \(f\) has limit \(L\) as the input \((x,y)\) approaches the ordered pair \((a,b)\) if the outputs of \(f\) approach \(L\) from all directions. In this case, we write
    \begin{equation*} \lim_{(x,y)\to (a,b)} f(x,y) = L \end{equation*}
    Formally, we say the limit exists if for every \(\varepsilon \gt 0\text{,}\) there exists a \(\delta \gt 0\) such that
    \begin{equation*} \sqrt{(x-a)^2 + (y-b)^2} \lt \delta \Rightarrow |f(x) - L| \lt \varepsilon \end{equation*}

    Fact 10.2.3.

    The limit laws from Calculus 1 apply to functions of multiple variables.

    Activity 10.2.1.

    Compute the following limits.
    (a)
    \(\ds \lim_{(x,y)\to (1,2)} \dfrac{5x^2}{x^2 + y^2}\)
    (b)
    \(\ds \lim_{(x,y)\to (11,13)} \sqrt{\dfrac{1}{xy}}\)

    Remark 10.2.4.

    For functions of one variable, there is only one possible way to approach a function: from the left or from the right. For functions of multiple variable, we can also approach a function along a path.

    Activity 10.2.2.

    Consider the function \(f(x,y) = \dfrac{7xy}{3x^2 + y^2}\text{.}\)
    (a)
    Why is it not possible to compute the limit \(\ds \lim_{(x,y)\to (0,0)} f(x,y)\text{?}\)
    (b)
    We can compute the limit if we approach along a specific line. Along which path are we approaching in the limit \(\ds \lim_{(x,y)\to (x,0)} f(x,y) \text{?}\)
    1. The line \(y = 0\)
    2. The line \(x = 0\)
    3. The line \(y = x\)
    4. The parabola \(y = x^2\)
    (c)
    Compute the limit from part (b).
    (d)
    Show the limit doesn’t exist by approaching along the line \(y = x\text{.}\) Which expression is the correct one to express this limit?
    1. \(\displaystyle \ds \lim_{(x,y) \to (0,0)} f(x,y)\)
    2. \(\displaystyle \ds \lim_{(x,y)\to (y,x)} f(x,y)\)
    3. \(\displaystyle \ds \lim_{(x,y)\to (x,x)} f(x,y)\)
    4. \(\displaystyle \ds \lim_{(x,y)\to (y,y)} f(x,y)\)

    Remark 10.2.5.

    Continuity for functions of multiple variables is defined similarly to functions of a single variable. We say that a function \(f(x,y)\) is continuous at the point \((a,b)\) if
    1. \(f(a,b)\) exists
    2. \(\ds \lim_{(x,y)\to (a,b)} f(x,y)\) exists
    3. \(\displaystyle \ds \lim_{(x,y)\to (a,b)} f(x,y) = f(a,b)\)

    Activity 10.2.3.

    Consider the function the function \(g(x,y) = \sqrt{26-2x^2-y^2}\)
    (a)
    True or false: \(g(x,y)\) is continuous at \((2,-3)\text{.}\)
    1. True
    2. False
    (b)
    For what inputs is the function discontinuous? Why?
