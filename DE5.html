    Section 2.5 DE5: The Product and Quotient Rules

    Objectives

    DE5
    I can use the product rule and the quotient rule to compute the derivative of products or quotients of functions.

    Pre-Class Preparation.

    Read the following sections of the text:
    • Section 3.3: Differentiation Rules
      Read the "The Product Rule", "The Quotient Rule", and "Combining Differentiation Rules" sections

    Subsection 2.5.1 In-Class Activities

    Activity 2.5.1.

    Let \(f(t) = 8t^5 -6\sqrt{t}\) and \(g(t) = 2t\text{.}\)
    (a)
    Compute \(f'(t)\) and \(g'(t)\)
    (b)
    (a)
    Find the product function, \(P(t) = f(t)\cdot g(t)\text{.}\)
    (b)
    Compute \(P'(t)\)
    (c)
    True or false: \(P'(t) = f'(t)\cdot g'(t)\)
    (c)
    (a)
    Find the quotient function, \(Q(t) = \dfrac{f(t)}{g(t)}\text{.}\)
    (b)
    Compute \(Q'(t)\)
    (c)
    True or false: \(Q'(t) = \dfrac{f'(t)}{g'(t)}\)

    Theorem 2.5.1. The Product Rule.

    Let \(f(x)\) and \(g(x)\) be differentiable functions. Then,
    \begin{equation*} \dfrac{d}{dx}[f(x)\cdot g(x)] = f'(x)g(x) + f(x)g'(x) \end{equation*}

    Theorem 2.5.2. The Quotient Rule.

    Let \(f(x)\) and \(g(x)\) be differentiable functions. Then,
    \begin{equation*} \dfrac{d}{dx}\left[\dfrac{f(x)}{g(x)}\right] = \dfrac{f'(x)g(x) - f(x)g'(x)}{(g(x))^2} \end{equation*}
    provided \(g(x)\neq 0\text{.}\)

    Activity 2.5.2.

    The product rule is a powerful tool, but sometimes it isn’t necessary; a more elementary rule may suffice. For which of the following functions can you find the derivative without using the product rule? Select all that apply and find the derivative.
    1. \(\displaystyle f(x)=e^x \sin(x)\)
    2. \(\displaystyle f(x)=\sqrt{x}(x^3+3x-3)\)
    3. \(\displaystyle f(x)=(4)(x^5)\)
    4. \(\displaystyle f(x)=x \ln(x)\)

    Activity 2.5.3.

    Use the product rule to find the derivative of the following functions.
    (a)
    \(f(x)=(x^2+3x)\tan(x)\)
    (b)
    \(f(x)=2^x \cos(x)\)
    (c)
    \(f(x) = x^2e^x\)

    Activity 2.5.4.

    Just like with the product rule, there are times when we can find the derivative of a quotient using elementary rules rather than the quotient rule. For which of the following functions can you find the derivative without using the quotient rule? Select all that apply and find the derivative.
    1. \(\displaystyle f(x) = \dfrac{6}{x^3}\)
    2. \(\displaystyle f(x) = \dfrac{-5}{\ln(x)}\)
    3. \(\displaystyle f(x) = \dfrac{e^x}{\csc(x)}\)
    4. \(\displaystyle f(x) = \dfrac{x^3+3x}{x}\)

    Activity 2.5.5.

    Find the derivative of the following functions using the quotient rule (or, if applicable, an elementary rule).
    (a)
    \(f(x) = \dfrac{\csc(x)}{x^3}\)
    (b)
    \(g(x) = \dfrac{x^2 + x^{5/3}}{x^4}\)
    (c)
    \(h(x) = \dfrac{5e^x - \cos(x)}{x}\)

    Activity 2.5.6.

    Use the quotient rule and appropriate trigonometric identities to prove the following:
    (a)
    \(\dfrac{d}{dx}[\tan(x)] = \sec^2(x)\)
    (b)
    \(\dfrac{d}{dx}[\cot(x)] = -\csc^2(x)\)
    (c)
    \(\dfrac{d}{dx}[\sec(x)] = \sec(x)\tan(x)\)
    (d)
    \(\dfrac{d}{dx}[\csc(x)] = -\csc(x)\cot(x)\)
