Theorem 10.5.7. Second Partial Derivative Test.

Let \(f(x,y)\) be a twice-differentiable function, and let \((x_0,y_0)\) be a critical point of \(f\text{.}\) Set
\begin{equation*} D = f_{xx}(x_0,y_0)f_{yy}(x_0,y_0) - (f_{xy}(x_0,y_0))^2 \end{equation*}
Then,
  1. If \(D\gt 0\) and \(f_{xx}(x_0,y_0) \gt 0\text{,}\) then \((x_0,y_0)\) is a local minimum of \(f\)
  2. If \(D\gt 0\) and \(f_{xx}(x_0,y_0) \lt 0\text{,}\) then \((x_0,y_0)\) is a local maximum of \(f\)
  3. If \(D \lt 0\text{,}\) then \((x_0,y_0)\) is a saddle point of \(f\)
  4. If \(D = 0\text{,}\) the test is inconclusive
