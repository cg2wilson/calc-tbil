Activity 1.4.2.

Use Remark 1.4.1 and Figure 1.4.2 to answer the following tasks.
(a)
\(f(x)\) is not continuous at \(x=0\text{.}\) Why? Select all that apply.
  1. \(\ds \lim_{x\to 0} f(x)\) does not exist
  2. \(\ds \lim_{x\to 0^-} f(x) = -3\) but \(\ds \lim_{x\to 0^+} f(x) = 0\)
  3. \(\ds \lim_{x\to 0^-} = 0\) but \(\ds \lim_{x\to 0^+} f(x) = -3\)
  4. \(f(x)\) has two outputs at \(x=0\)
(b)
\(f(x)\) is not continuous at \(x=1\text{.}\) Why? Select all that apply.
  1. \(f(1)\) and \(\ds \lim_{x\to 1} f(x)\) are not the same
  2. \(\displaystyle \ds \lim_{x\to 1^+} f(x) = +\infty\)
  3. \(\displaystyle \ds \lim_{x\to 1^+} f(x)\neq \lim_{x\to 1^-} f(x)\)
  4. \(\ds \lim_{x\to 1} f(x)\) does not exist
(c)
\(f(x)\) is not continuous at \(x=3\text{.}\) Why? Select all that apply.
  1. \(\ds \lim_{x\to 3} f(x)\) does not exist
  2. \(f(3)\) and \(\ds \lim_{x\to 3} f(x)\) are not the same
  3. \(f(x)\) has two outputs at \(x=3\)
  4. \(\displaystyle \ds \lim_{x\to 3^-} f(x) = \lim_{x\to 3^+} f(x)\)
