(a)
Why are we unable to use the average rate of change formula to answer part (d) of Activity 2.1.3?
  1. We are actually able to use the average rate of change formula, we just didn’t use it there.
  2. If we computed it using \(t_1=t_2=2\text{,}\) we would end up with \(\dfrac{0}{0}\) for the average rate of change.
  3. We don’t have specific numbers to put into the formula.
  4. If we computed it using \(t_1=t_2=2\text{,}\) only the denominator would be \(0\text{,}\) and that makes the fraction undefined.
