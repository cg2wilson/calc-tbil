Activity 7.3.2.

Consider the triangle below.
(a)
If \(x = a\tan\theta\text{,}\) then
  1. \(\displaystyle A=a,B=x,C=\sqrt{x^2+a^2}\)
  2. \(\displaystyle A=x,B=a,C=\sqrt{x^2+a^2}\)
  3. \(\displaystyle A=a,B=x,C=x^2+a^2\)
  4. \(\displaystyle A=x,B=a,C=x^2+a^2\)
(b)
If \(x=a\sin\theta\text{,}\) then
  1. \(\displaystyle A=x,B=a,C=x^2+a^2\)
  2. \(\displaystyle A=x,B=a,C=\sqrt{a^2-x^2}\)
  3. \(\displaystyle A=\sqrt{a^2-x^2},B=x,C=a\)
  4. \(\displaystyle A=\sqrt{x^2-a^2},B=x,C=a\)
(c)
If \(x=a\sec\theta\text{,}\) then
  1. \(\displaystyle A=a,B=x,C=\sqrt{x^2+a^2}\)
  2. \(\displaystyle A=x,B=\sqrt{x^2-a^2},C=a\)
  3. \(\displaystyle A=x,B=a,C=\sqrt{x^2+a^2}\)
  4. \(\displaystyle A=a,B=\sqrt{a^2-x^2},C=x\)
