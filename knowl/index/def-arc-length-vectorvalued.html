Definition 12.2.5. Arc Length (Vector-Valued Function).

Consider the smooth vector valued function \(\textbf{r}(t) = \lrangle{f(t),g(t)}\text{,}\) defined on the interval \([a,b]\text{.}\) The arc length of \(\textbf{r}(t)\) on \([a,b]\) is given by
\begin{equation*} s = \int_a^b \sqrt{[f'(t)]^2 + [g'(t)]^2}\, dt = \int_a^b ||\textbf{r}'(t)||\, dt \end{equation*}
Arc length is defined similarly for a space curve of three components.
