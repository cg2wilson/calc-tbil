Definition 11.3.1. Triple Integral (Cartesian Coordinates).

The triple integral of a function \(f(x,y,z)\) over a rectangular box \(B\) is defined to be
\begin{equation*} \ds \lim_{l,m,n\to\infty} \sum_{i=1}^l\sum_{j=1}^m\sum_{k=1}^n f(x_{ijk},y_{ijk},z_{ijk})\Delta x \Delta y \Delta z = \iiint_B f(x,y,z)\, dV \end{equation*}
