Definition 5.3.6. Scalar Equation of a Plane.

The scalar equation of a plane containing the point \(P(x_0,y_0,z_0)\) with normal vector \(\textbf{n} = \lrangle{a,b,c}\) is given by
\begin{equation*} a(x-x_0) + b(y-y_0) + c(z-z_0) = 0 \end{equation*}
or alternately,
\begin{equation*} ax + by + cz + d = 0 \end{equation*}
