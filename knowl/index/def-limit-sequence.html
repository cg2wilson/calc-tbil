Definition 8.1.4. Limit of a Sequence.

Given a sequence \(\lrbrace{a_n}\text{,}\) if the terms \(a_n\) become arbitrarily close to a finite number \(L\) as \(n\) becomes sufficiently large, we say \(\lrbrace{a_n}\) is a convergent sequence and that \(L\) is the limit of the sequence. We write this as
\begin{equation*} \lim_{n\to\infty} a_n = L \end{equation*}
If a sequence is not convergent, it is divergent
