Definition 12.2.1. Vector-Valued Function.

A vector-valued function is a function of the form
\begin{equation*} \textbf{r}(t) = f(t)\textbf{i} + g(t)\textbf{j} \text{ or } \textbf{r}(t) = f(t)\textbf{i} + g(t)\textbf{j} + h(t)\textbf{k} \end{equation*}
where the component functions \(f(t), g(t), h(t)\) are real-valued functions of the parameter \(t\text{.}\)
