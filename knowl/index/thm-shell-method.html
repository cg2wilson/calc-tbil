Theorem 6.3.3. Shell Method.

Let \(f(x)\) be a continuous function on the interval \([a,b]\text{.}\) Then, the volume of the solid created by rotating the region bounded \(f(x)\) and the \(x-axis\) about the line \(x = 0\) is given by
\begin{equation*} \ds \int_a^b 2\pi r(x)f(x)\, dx \end{equation*}
where \(r(x)\) is a radius function describing the distance from \(x=0\) to the cylinder.
