Definition 8.1.1. Sequence.

A sequence is an ordered list of numbers of the form
\begin{equation*} a_1,a_2,a_3,...,a_n,... \end{equation*}
Each value in the sequence is called a term. The symbol \(n\) is called the index for the sequence. We denote a sequence by
\begin{equation*} \lrbrace{a_n}_{n=i}^j \end{equation*}
when the sequence begins at index value \(i\) and ends at index value \(j\text{.}\) If the sequence is infinite, we instead write
\begin{equation*} \lrbrace{a_n}_{n=1}^\infty \end{equation*}
or
\begin{equation*} \lrbrace{a_n} \end{equation*}
