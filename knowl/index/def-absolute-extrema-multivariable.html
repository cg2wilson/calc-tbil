Definition 10.5.5. Absolute Max/Min (Multivariable).

Let \(z = f(x,y)\) be a function of two variables that is defined and continuous on an open set containing the point \((x_0,y_0)\text{.}\) We say that \(f\) has an absolute maximum at \((x_0,y_0)\) if
\begin{equation*} f(x_0,y_0)\geq f(x,y) \end{equation*}
for all points \((x,y)\) in the domain of \(f\text{.}\) \(f\) has an absolute minimum at \((x_0,y_0)\) if
\begin{equation*} f(x_0,y_0) \leq f(x,y) \end{equation*}
for all points \((x,y)\) in the domain of \(f\text{.}\)
