Definition 13.1.3. Curl.

For a vector field \(\textbf{F} = \lrangle{P,Q,R}\) for which the first partials of each component exists, we define the curl of \(\textbf{F}\) to be
\begin{equation*} \curl{F} = \lrangle{R_y-Q_z,P_z-R_x,Q_x-P_y} = \left|\begin{matrix} \textbf{i} \amp \textbf{j} \amp \textbf{k} \\ \dfrac{\partial}{\partial x} \amp \dfrac{\partial}{\partial y} \amp \dfrac{\partial}{\partial z} \\ P \amp Q \amp R \end{matrix}\right| \end{equation*}
In terms of the del operator, we can write
\begin{equation*} \curl{F} = \nabla \times\textbf{F} \end{equation*}
