Definition 8.1.8. Monotonic Sequence.

A sequence is said to be monotonically increasing for all \(n\geq N\) if \(a_n\leq a_{n+1}\) for all \(n\geq N\text{;}\) if \(a_n \geq a_{n+1}\) for all \(n\geq N\text{,}\) then the sequence is said to be monotonically decreasing for all \(n\geq N\text{.}\)
A sequence is said to be monotonic or monotone if it is monotonically increasing or monotonically decreasing.
