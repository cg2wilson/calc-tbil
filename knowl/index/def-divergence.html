Definition 13.1.1. Divergence.

For a vector field \(\textbf{F} = \lrangle{P,Q,R}\text{,}\) if each component function is differentiable, the divergence of \(\textbf{F}\) is given by
\begin{equation*} \divr{F}= P_x + Q_y + R_z \end{equation*}
In terms of the del operator
\begin{equation*} \nabla = \lrangle{\dfrac{\partial}{\partial x},\dfrac{\partial}{\partial y},\dfrac{\partial}{\partial z}} \end{equation*}
we can express the divergence of a vector field as
\begin{equation*} \divr{F} = \nabla\cdot\textbf{F} \end{equation*}
The definition of divergence can be extended to a vector in \(\R^n\) in the natural way.
