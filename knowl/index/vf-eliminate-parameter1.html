Activity 12.1.4.

We can convert between parametric and rectangular forms of equations through a process called eliminating the parameter.
(a)
Consider the parametrization \(x(t) = t+1\text{,}\) \(y = t^2 + 1\) on \(-\infty\leq t \leq -1\text{.}\) Which of the following shows the appropriate rectangular equation (and interval) for the parametric curve?
  1. \(y = (x-1)^2 + 1\) on \(-\infty\leq x\leq 1\)
  2. \(y = (x+1)^2 + 1\) on \(-\infty\leq x\leq 0\)
  3. \(y = (x+1)^2 + 1\) on \(-\infty\leq x\leq -1\)
  4. \(y = (x-1)^2 + 1\) on \(-\infty\leq x\leq 0\)
Answer.
A
(b)
Eliminate the parameter to find a Cartesian description for the parametric equation \(x(t) = \cos (2\pi t)\text{,}\) \(y = \sin (2\pi t)\text{,}\) on \(0\leq t \leq 1\text{.}\) Think of equations relating sine and cosine...
