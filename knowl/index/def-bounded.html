Definition 8.1.6. Bounded Sequence.

A sequence \(\lrbrace{a_n}\) is bounded above if there exists a real number \(M\) such that \(a_n \leq M\) for every \(n\text{.}\) \(\lrbrace{a_n}\) is bounded below if there exists a real number \(N\) such that \(N \leq a_n\) for every \(n\text{.}\)
If a sequence is bounded above and bounded below, we say it is bounded. If a sequence is not bounded (above or below), it is said to be unbounded.
