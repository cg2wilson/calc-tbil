Definition 12.3.4. Line Integral (Vector).

Let \(\textbf{F}\) be a continuous vector field defined on some smooth curve \(C\) given by the vector function \(\textbf{r}(t), a\leq t\leq b\text{.}\) Then, the line integral of F along C is given by
\begin{equation*} \int_C \textbf{F}\cdot d\textbf{r} = \int_a^b \textbf{F}(\textbf{r}(t))\cdot \textbf{r}'(t)\, dt = \int_C \textbf{F}\cdot \textbf{T}\, ds \end{equation*}
where \(\textbf{T}\) is the unit tangent vector at a given point.
