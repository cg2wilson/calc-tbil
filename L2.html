    Section 1.2 L2: Limits Analytically

    Objectives

    L2
    I can use limit laws and algebraic techniques to compute the one-sided or two-sided limit of a function.

    Pre-Class Preparation.

    Read the following sections of the text:
    • Section 2.3: The Limit Laws
      Do not read "The Squeeze Theorem"

    Subsection 1.2.1 In-Class Activities

    Remark 1.2.1.

    Recall that in Activity1.1.10 we used numerical methods and table of values to find the limit of a relatively simple degree three polynomial at a point. This was inefficient, theres gotta be a better way!

    2. \(\displaystyle \ds \lim_{x \to 2}f(x) \approx f(2)\)
    3. \(\displaystyle \ds \lim_{x \to 2}f(x) \neq f(2)\)

    Theorem 1.2.3. Limit Laws, I.

    Let \(f\) and \(g\) be functions defined on an open interval \(I\) containing the real number \(c\) satisfying
    \begin{equation*} \lim_{x \to c} f(x) = L \, \text{and} \, \lim_{x \to c} g(x) = K, \end{equation*}
    for \(L\) and \(K\) some real numbers. Then we have the following limits.
    1. Constant Law: \(\ds \lim_{x \to c} b = b\text{,}\) for \(b\) any constant real number
    2. Identity Law: \(\ds \lim_{x \to c} x = c\)
    3. Sum/Difference Law: \(\ds \lim_{x \to c} (f(x) \pm g(x)) = L \pm K\)
    4. Scalar Multiple Law: \(\ds \lim_{x \to c} [b \cdot f(x)] = bL\text{,}\) for \(b\) any constant real number
    5. Product Law: \(\ds \lim_{x \to c} [f(x) \cdot g(x)] = LK\)
    6. Quotient Law: if \(K \ne 0\text{,}\) then \(\ds \lim_{x \to c} [f(x) / g(x)] = L/K\text{.}\)

    Activity 1.2.3.

    If \(\ds \lim_{x\to 2} f(x) = 2\) and \(\ds \lim_{x\to 2} g(x) = -3\text{,}\) which of the following statements are true? Select all that apply!
    1. \(\displaystyle \ds \lim_{x\to 2} (f(x) \cdot g(x)) = -6\)
    2. \(\displaystyle \ds \lim_{x\to 2} (f(x) + g(x)) = -1\)
    3. \(\displaystyle \ds \lim_{x\to 2} (f(x) - g(x)) = -2\)
    4. \(\displaystyle \ds \lim_{x\to 2} (f(x)/g(x)) = -2/3\)

    Theorem 1.2.4. Limit Laws, II.

    Let \(f\) and \(g\) be functions defined on an open interval \(I\) containing \(c\) satisfying
    \begin{equation*} \lim_{x \to c} f(x) = L, \, \, \, \, \lim_{x \to L} g(x) = K, \text{and} \, g(L)=K. \end{equation*}
    Then we have the following limits as well.
    1. Power Law: \(\ds \lim_{x \to c} [f(x)]^n = L^n\text{,}\) for \(n\) a positive integer
    2. Root Law: \(\ds \lim_{x \to c} \sqrt[n]{f(x)} = \sqrt[n]{L}\text{,}\) for \(n\) a positive integer
    3. Composition Law: \(\ds \lim_{x \to c} g(f(x)) = K\)

    Activity 1.2.5.

    Given \(p(x) = -3x^2 - 5x+7\text{,}\) which of the following limit laws would use to determine \(\ds \lim_{x\to 2} p(x)\text{?}\) Choose all that apply.
    1. Sum/Difference Rule
    2. Constant Multiple Law
    3. Multiplication Law
    4. Constant Law
    5. Exponent Law
    6. Constant Law
    7. Division Law

    Theorem 1.2.6. Direct Substitution Property.

    If \(f\) is a function with \(a\) in the domain of \(f\text{,}\) then
    \begin{equation*} \ds \lim_{x\to a} f(x) = f(a) \end{equation*}
    provided that \(f\) is not a piecewise function.

    Activity 1.2.6.

    Given \(p(x)=-3x^2-5x+7\) and \(q(x)=x^4-x^2+3\text{,}\) which of the following describes the most efficient way to determine \(\ds \lim_{x \to -1} \dfrac{p(x)}{q(x)}\text{?}\)
    1. Sum/difference, Constant Multiple, and Multiplication Laws
    2. Theorem1.2.6 and the Division Law
    3. Exponent, Sum/difference, Constant Multiple, and Constant Laws
    4. Division and Root Laws

    Activity 1.2.7.

    Consider taking the limit of a rational function \(\dfrac{p(x)}{q(x)}\) as \(x \to c\text{.}\) If \(q(c)=0\text{,}\) is it possible for \(\ds\lim_{x \to c}\dfrac{p(x)}{q(x)} \) to equal a number?
    1. No, because \(\dfrac{p(x)}{q(x)}\) is not defined at \(x=c\) since \(q(c)=0\)
    2. Yes, because if you graph \(f(x)=\dfrac{x^2-1}{x-1}\text{,}\) the value \(f(1)\) is not defined, but the graph shows that the limit of \(f(x)\) does exist as \(x \to 1\)
    3. No, because if you graph \(g(x)=\dfrac{x^2+1}{x-1}\text{,}\) the value \(g(1)\) is not defined and the graph shows that the limit of \(\displaystyle\lim_{x \to c}g(x)\) does not exist
    4. Yes, because we can use Theorem1.2.6

    Activity 1.2.8.

    Let \(f(x) = 2x\) and \(g(x) = x\text{,}\) which of the following statements is true?
    1. \(\displaystyle \ds \lim_{x\to 0} (f(x)/g(x)) = 0\)
    2. \(\displaystyle \ds \lim_{x\to 0} (f(x)/g(x)) = 2\)
    3. \(\ds \lim_{x\to 0} (f(x)/g(x))\) cannot be determined
    4. \(\ds \lim_{x\to 0} (f(x)/g(x))\) does not exist

    Remark 1.2.7.

    When we compute the limit of a ratio where both the numerator and denominator have limit equal to zero, we have to compute the value of a \(\dfrac{0}{0}\) indeterminate form. The value of an indeteminate form can be any real number or even infinity or not existent, we just do not know yet! We can usually determine the value of an indeterminate form using some algebraic manipulations of the expression given.

    Definition 1.2.8.

    A function \(f(x)\) has a hole at \(x=c\) if \(f(c)\) does not exist but \(\ds \lim_{x \to c} f(x)\) does exist and is equal to a real number.
