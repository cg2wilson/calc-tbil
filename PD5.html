    Section 10.5 PD5: Multivariate Optimization

    Objectives

    PD5
    I can locate and classify the critical points of a function of two variables using the Second Derivative Test, and find the absolute extrema for a surface.

    Pre-Class Preparation.

    Read the following sections of the text:
    • Section 4.7: Maxima/Minima Problems
      Read all sections

    Subsection 10.5.1 In-Class Activities

    Remark 10.5.1.

    Recall that a critical point in Calculus 1 is found by setting the derivative equal to zero, i.e. \(f'(x) = 0\)

    Definition 10.5.2. Critical Point (Multivariable Version).

    Let \(z=f(x,y)\) be a function of two variables. The point \((x_0,y_0)\) is a critical point of \(f(x,y)\) if either
    1. \(\nabla f(x_0,y_0) = \textbf{0}\text{,}\) i.e. \(f_x(x_0,y_0) = f_y(x_0,y_0) = 0\)
    2. Either \(f_x(x_0,y_0)\) or \(f_y(x_0,y_0)\) does not exist

    Activity 10.5.1.

    Let \(f(x,y) = x^3+2xy-2x-4y\text{.}\)
    (a)
    Identify \(f_x(x,y)\) and \(f_y(x,y)\)
    Solution.
    \(f_x = 3x^2 + 2y -2\) and \(f_y = 2x-4\)
    (b)
    In order to begin finding the critical points, which expression(s) is the most appropriate to evaluate?
    1. \(\displaystyle 3x^2+2y-2 = 0\)
    2. \(\displaystyle 3x^2 + 2y-2 = 2x-4\)
    3. \(\displaystyle 2x-4 = 0\)
    4. \(\displaystyle 3x^2 + 2(2x-4) - 2 = 0\)
    Solution.
    C
    (c)
    Solving the equation from (c) gives what conditions on the variable(s)?
    Solution.
    \(x = 2\)
    (d)
    Plug the condition(s) from (c) into the other partial derivative. What critical point(s) do you get?
    Solution.
    If \(x = 2\text{,}\) then \(y = -5\text{,}\) which gives only one critical point of \((2,-5)\)

    Activity 10.5.2.

    Argue why the only critical point of the function \(g(x,y) = x^2+xy+y^2+2x-2y+5\) is \((-2,2)\)
    Solution.
    \(\nabla g = \lrangle{2x+y+2,x+2y-2}\text{.}\) Setting \(f_x = 0\text{,}\) we get \(y = -2x-2\text{.}\) Plugging this into \(f_y = 0\text{,}\) we get \(x + 2(-2x-2)-2 = 0\text{,}\) which gives \(x = -2\text{.}\) Since \(x = -2, y = -2(-2)-2 = 2\text{,}\) giving the desired critical point.

    Remark 10.5.3.

    The definitions of local and absolute extrema for multivariable functions are analogous to the definitions for single-variable functions.

    Definition 10.5.4. Local Max/Local Min (Multivariable).

    Let \(z = f(x,y)\) be a function of two variables that is defined and continuous on an open set containing the point \((x_0,y_0)\text{.}\) We say that \(f\) has a local maximum at \((x_0,y_0)\) if
    \begin{equation*} f(x_0,y_0)\geq f(x,y) \end{equation*}
    for all points \((x,y)\) within some disk centered at \((x_0,y_0)\text{.}\) \(f\) has a local minimum at \((x_0,y_0)\) if
    \begin{equation*} f(x_0,y_0) \leq f(x,y) \end{equation*}
    for all points \((x,y)\) within some disk centered at \((x_0,y_0)\text{.}\)

    Definition 10.5.5. Absolute Max/Min (Multivariable).

    Let \(z = f(x,y)\) be a function of two variables that is defined and continuous on an open set containing the point \((x_0,y_0)\text{.}\) We say that \(f\) has an absolute maximum at \((x_0,y_0)\) if
    \begin{equation*} f(x_0,y_0)\geq f(x,y) \end{equation*}
    for all points \((x,y)\) in the domain of \(f\text{.}\) \(f\) has an absolute minimum at \((x_0,y_0)\) if
    \begin{equation*} f(x_0,y_0) \leq f(x,y) \end{equation*}
    for all points \((x,y)\) in the domain of \(f\text{.}\)

    Definition 10.5.6. Saddle Point.

    A function \(f(x,y)\) has a saddle point at \((x_0,y_0)\) if \(\nabla f(x_0,y_0) = \textbf{0}\) but \((x_0,y_0)\) is neither a maximum nor a minimum.

    Activity 10.5.3.

    Consider the function \(f(x,y) = y^2-x^2\text{.}\)
    (a)
    Find the critical point for \(f(x,y)\text{.}\)
    Solution.
    The critical point is at \((0,0)\)
    (b)
    Consider the traces \(f(0,y)\) and \(f(x,0)\text{.}\) These traces show that
    1. the critical point is a maximum for both traces, thus a maximum
    2. the critical point is a minimum for both traces, thus a minimum
    3. the critical point is a maximum for one trace, and a minimum for the other, and thus a saddle point
    Solution.
    C

    Theorem 10.5.7. Second Partial Derivative Test.

    Let \(f(x,y)\) be a twice-differentiable function, and let \((x_0,y_0)\) be a critical point of \(f\text{.}\) Set
    \begin{equation*} D = f_{xx}(x_0,y_0)f_{yy}(x_0,y_0) - (f_{xy}(x_0,y_0))^2 \end{equation*}
    Then,
    1. If \(D\gt 0\) and \(f_{xx}(x_0,y_0) \gt 0\text{,}\) then \((x_0,y_0)\) is a local minimum of \(f\)
    2. If \(D\gt 0\) and \(f_{xx}(x_0,y_0) \lt 0\text{,}\) then \((x_0,y_0)\) is a local maximum of \(f\)
    3. If \(D \lt 0\text{,}\) then \((x_0,y_0)\) is a saddle point of \(f\)
    4. If \(D = 0\text{,}\) the test is inconclusive

    Activity 10.5.4.

    Let \(f(x,y) = x^2+y^2-2x-6y+14\text{.}\)
    (a)
    Find the critical point(s) of \(f\)
    Solution.
    The critical point is \((1,3)\)
    (b)
    Classify the critical point(s) of \(f\) using Theorem 10.5.7
    Solution.
    \(f_{xx} = 2\text{,}\) \(f_{yy} = 2\text{,}\) \(f_{xy} = 0\text{.}\) So,
    \begin{equation*} D = (2)(2) - 0^2 = 4 \end{equation*}
    since \(f_{xx} = 2 \gt 0\text{,}\) \((1,3)\) is a minimum for the function.

    Activity 10.5.5.

    Find and classify the critical points of the following functions.
    (a)
    \(f(x,y) = -x^3+4xy-2y^2+1\)
    Solution.
    The critical points \((0,0)\) and \(\lrpar{-\dfrac{4}{3},-\dfrac{4}{3}}\) are saddle points
    (b)
    \(f(x,y) = x^2y^2\)
    Solution.
    The second partial derivative test is inconclusive about the critical point \((0,0)\text{.}\) However, we can conclude it is a minimum from the form of the function.

    Remark 10.5.8.

    In Calculus I, finding the absolute extrema of a function means we need to check the function value at the boundary of the interval; this is just the two endpoints. For a two-variable function, we need to check the boundary of the entire domain.

    Finding Absolute Extrema for Functions of Two Variables.

    1. Determine the critical points of the function \(f(x,y)\)
    2. Compute the function value for each critical point
    3. Compute the maximum value along the boundary of the domain. This may mean you need to split the boundary into separate pieces of analysis.
    4. The largest and smallest values in steps 2 and 3 are the absolute maximum and minimum, respectively.

    Activity 10.5.6.

    Let \(f(x,y) = 2+2x+4y-x^2-y^2\text{.}\) We will find the absolute extrema on the triangular region in the first quadrant bounded by the lines \(x=0\text{,}\) \(y = 0\text{,}\) and \(y = 9-x\)
    (a)
    Identify the critical point of the function on the domain.
    Solution.
    \((1,2)\)
    (b)
    What is the value of \(f\) at the critical point?
    Solution.
    7
    (c)
    To evaluate along the lower portion of the triangle that forms the domain, which equation should be used?
    1. \(\displaystyle y=0\)
    2. \(\displaystyle x=0\)
    3. \(\displaystyle y=9-x\)
    4. \(\displaystyle y = 9\)
    Solution.
    A
    (d)
    Evaluate the function along all three sides
    Solution.
    Along \(y = 0\text{,}\) we have \(f(x,0) = 2+2x-x^2\text{.}\) \(f'(x,0) = 2-2x\text{,}\) which assumes a maximum at \(x = 1\text{.}\) This gives the output \(f(1,0) = 3\text{.}\)
    Along \(x = 0\text{,}\) we have \(f(0,y) = 4y-y^2\text{.}\) \(f'(0,y) = 4-2y\text{,}\) which assumes a maximum at \(y = 2\text{.}\) This gives the output \(f(0,2) = 6\)
    Along \(y = 9-x\text{,}\) we have \(f(x,9-x) = -43+16x-2x^2\text{.}\) \(f'(x,9-x) = 16-4x\text{,}\) which assumes a maximum at \(x = 4\text{.}\) This gives the output \(f(4,5) = -11\)
    (e)
    Evaluate the function at the three corners.
    Solution.
    At \((0,0)\text{,}\) \(f(0,0) = 2\)
    At \((9,0)\text{,}\) \(f(9,0) = -61\)
    At \((0,9)\text{,}\) \(f(0,9) = -43\)
    (f)
    What is the absolute maximum? What about the absolute minimum?
    Solution.
    The absolute maximum is at \((1,2,7)\) and the absolute minimum is at \((9,0,-61)\)

    Activity 10.5.7.

    Identify the absolute extrema of \(f(x,y) = x^2-2xy+2y\) on the rectangle \(0\leq x\leq 3\text{,}\) \(0\leq y\leq 2\)
    Solution.
    The absolute maximum is at \((3,2,25)\text{,}\) and the absolute minimum is at \((0,0,0)\)
